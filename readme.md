# Shooter
Цель проекта - реализовать прототип уровня, в котором персонаж бежит из одной точки в другую и при этом сталкивается с противниками и физическими преградами.

## Реализовано:
1. Управление персонажем (движение, присед, прыжок, прицеливание, стрельба)
2. Анимации персонажа (на все состояния из п.1), в т.ч риггинг
3. Противник дрон, который патрулирует зону и стреляет по персонажу, если его заметит
4. Противник солдат, который может как патрулировать зону, так и преследовать, искать и стрелять в персонажа. Анимации схожи с анимацией персонажа
5. Транспортное средство. Управление, движение, стрельба
6. Система взаимодействия с предметами (набрать код двери, забраться по лестнице, подобрать аптечку)
7. Система квестов. Два типа: завершение при вхождении в триггер или убийстве всех врагом в области
8. Есть урон от падения
9. При смерти есть возможность возродиться на месте завершения предыдущего квеста
10. Сделан план уровня, затем greybox, затем whitebox
11. Звуковое сопровождение (фоновая музыка, бег, стрельба, попадание снаряда, шум воды, звук мотора машины)
12. Разная визуализация и звук от попадания по поверхности в зависимости от материала поверхности (камень, металл, песок, древесина). Аналогично и с бегом по разным материалам
